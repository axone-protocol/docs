## axoned debug prefixes

List prefixes used for Human-Readable Part (HRP) in Bech32

### Synopsis

List prefixes used in Bech32 addresses.

```
axoned debug prefixes [flags]
```

### Examples

```
$ axoned debug prefixes
```

### Options

```
  -h, --help   help for prefixes
```
